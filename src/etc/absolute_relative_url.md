# 절대경로 상대경로
```admonish info title="절대경로"
절대경로는 고유한경로이며  
최상위루트 에서 찾아가고자하는 위치까지 명시한 경로이다.
- 보통 '/' 부터 시작한다.
```
```admonish info title="상대경로"
상대경로는 현재 위치를 기준으로 찾아가고자하는 위치까지 명시한 경로이다.
- './'는 현재위치부터 시작한다는 의미이다.
- '../'는 상위 디렉토리로 이동한다는 뜻이다.
- './[CURRENT]/'는 현재 위치 하위디렉토리부터 시작한다는 의미이다.
```
